This is a World Baseball Classic Dashboard for Home Assistant.

It includes all Pools (clickeable) information and the Schedule of each game.

See for yourself in the screenshots below.

All you have to do is copy the wbc2026-card.html file into your /config/www/ folder.

Then create a new dashboard (I used Panel (Single Card)), add a new card and copy the code in the wbccard file.

That's it.
